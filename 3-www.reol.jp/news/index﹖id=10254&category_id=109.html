    「Reol Japan Tour 2020ハーメルンの大号令 -出港編-」神奈川県民ホール 振替公演　払い戻しに関して

    先日中止を発表いたしました、8月21日(金)「Reol Japan Tour 2020ハーメルンの大号令 -出港編-」（※4月4日振替公演）神奈川県民ホールの払い戻しに関してお知らせいたします。

    ●ローソンチケットでご購入のお客様

    払戻し期間：5月15日(金) 10:00〜6月15日(月) 23:59 

     

    ※FC先行で4/4公演のチケットご購入されたお客様

    お手元のチケットにて払戻しの手続きをローソン・ミニストップ店内のLoppiで行って下さい。

    その際、チケット金額のみ払戻しさせていただきます。

    払戻し期間終了後、配送料含めた全額のお客様手数料の払戻し券をお送りしますので届き次第、再度、ローソン・ミニストップ店内のLoppiで払戻しの手続きを行って下さい。

     

    ※FC先行で8/21公演のチケットご購入されたお客様

    払戻しの方法をローチケより申込時登録のメールアドレスへお送りします。そちらを必ずご確認ください

     

     

    ※FC先行以外でご購入されたお客様

    チケットはお引取いただいたコンビニエンスストアであれば、どの店舗でも払戻可能です。払戻し手続きは、ローソン・ミニストップ店内のLoppiで行って下さい。

     電子チケット購入いただいた方はローチケからのメールをご確認ください。

     

    ※Loppi操作方法の詳細は下記ＵＲＬよりご確認ください。

    http://l-tike.com/oc/lt/haraimodoshi/

    （注１）チケットの半券がもぎられているものについては、ご返金対応を致しかねますので予めご了承ください。

    （注２）チケット代以外の外手数料は全額ご返金させて頂きます。その際チケット購入時店頭にてお渡しした、各種手数料券が必要となります。もし各種手数料券を紛失された場合、外手数料のご返金は致しかねますので何卒ご了承ください。

     

    お問合せ：ローソンチケット　0570-000-777（10:00～18:00）

     

    ●チケットぴあにてご購入のお客様

    払い戻し期間： 5月15日(金) 10:00〜7月31日(金) 23:59

    払い戻し期間中の各店舗営業時間内にて払戻しをお願い致します。

    払戻し受付期間を過ぎますと、払戻しは出来なくなりますのでご注意下さい。

     

    チケットの引取方法などにより手続きが異なります。

    お客様のお引き取り方法をご確認の上、以下のURLから該当の払戻し方法の説明ページにお進みください。

     

    http://t.pia.jp/guide/refund.jsp　（PC/携帯共通）

     

    お問合せ：チケットぴあインフォメーション　 http://t.pia.jp/help/index.jsp

     

    ●イープラスにてご購入のお客様

    払い戻し期間： 5月15日(金) 10:00〜6月15日(月) 23:59

    ※払戻し方法は、チケットの受取方法や支払方法により異なります。　

    ※下記の払戻し方法チャートをご確認のうえ、払戻し期間内にお手続きをお願いします。

     

    払戻方法　http://eplus.jp/refund1/

    e+お客さまサポートページ　http://support.eplus.jp/

     

    お問合せ：イープラス　0570-06-9911（10：00-18：00)

     

     

    ●LINEチケットにてご購入のお客様

    払い戻し期間：5月15日(金) 10:00〜6月15日(月) 23:59 

    払戻については、別途「LINEチケットinfo」のトーク画面から案内が届きますので、指定の期間内に振込先をご登録ください。（期間内にご登録のない場合、払い戻しの対応ができませんのでご注意ください）なお、ご案内はチケットの購入者ではなく、保有者に払い戻しの通知が届きます。

    ※詳細はこちらよりご確認下さい。⇒

    https://help2.line.me/ticketapp/web/categoryId/20005226/pc?lang=ja
