    「第一回 秘密結社L社員総会」チケット先着販売実施決定！

    7月16日（月･祝）開催「第一回 秘密結社L 社員総会」チケットについて若干枚数の先着販売が決定いたしました！

    ◆「第一回 秘密結社L 社員総会」先着販売◆
    受付期間：7月10日（火）12:00～7月11日（水）23:59
    対象者：受付期間及び開催日に「秘密結社L」社員（要有料会員登録）の方
    購入可能枚数：１会員１枚まで

    「秘密結社L 第一回社員総会」へご来場の皆様は、ドレスコード "BLACK（黒）" を推奨しております。とびっきりのブラックコーディネートでのご来場を心よりお待ちしております！

    ※枚数には限りがあります。規定枚数に達し次第受付終了となりますので予めご了承ください。
    ※電子チケットではありません。
    ※チケットは、公演当日の開場時間以降に会場受付にて引き換え可能です。その際、「秘密結社L」社員確認が出来るご本人様のスマートフォンとお支払い完了時に送られるメール本文、顔写真付き身分証(無い方はお名前の確認出来るものを２点準備して下さい。)をお持ちください。不足があった場合はご入場いただけませんのでご注意ください。
    また、開演間際は混み合う可能性がございますので、開演時間に余裕をもってお越しください。
    ※座席番号は選べません。
    ※ご応募はご本人のみ可能です。ご本人以外の代理応募は受け付けておりません。既にチケットをお持ちの方は重複所持は出来かねますのでご注意ください。尚、代理応募が発覚した際にはチケットは無効とさせていただきます。
