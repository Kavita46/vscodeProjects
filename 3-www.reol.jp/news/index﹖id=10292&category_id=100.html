    アニメ「デジモンアドベンチャー：」EDテーマ＆初の映像作品リリース！

    テレビアニメ「デジモンアドベンチャー：」のエンディングテーマとして話題の新曲「Q?」を11月4日に配信リリース。さらに、12月23日に初の映像作品「Reol LIVE 2019-2020　-ハーメルンの大号令 / 侵攻アップグレード-」をDVDとBlu-rayでリリースすることが決定しました！

    Reolにとってキャリア初の映像作品となる今作には、昨年開催された「Reol Oneman Live 2019 侵攻アップグレード」よりツアーファイナルにあたる新木場STUDIO COAST公演から本人がセレクトしたライヴシーンと、今年8月16日にLINE CUBE SHIBUYAで行われた配信ライヴ 「ハーメルンの大号令 -接続編-」の模様がノーカットで完全収録されるほか、特典映像が収められます。
    また、本編のエンドロールには「ハーメルンの大号令 -接続編-」の生配信時にスーパーチャットにご協力いただいた視聴者のお名前がクレジットされることも決定しております。

    パッケージは通常盤DVDおよびBlu-ray、ファンクラブ限定盤Blu-rayの3形態。
    ファンクラブ限定盤は特殊パッケージ仕様となっており、フォトブックと両ツアーのタイトルのロゴを象ったキーチャームが同梱されます。なお、限定盤は完全受注生産となっており、申し込み受付期間は本日10月16日から11月3日23時59分までとなっておりますので、気になる人は早めにチェックしてください！
    さらに対象店舗で、いずれかを予約もしくは購入すると、先着で特典音源ディスクがプレゼントされます。こちらの特典内容の詳細は後日アナウンスされますので、続報をお楽しみに！
