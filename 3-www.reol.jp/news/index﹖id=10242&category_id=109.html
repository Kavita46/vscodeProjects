    「Reol Japan Tour 2020 ハーメルンの大号令」 振替公演並びに払い戻しのご案内

    新型コロナウイルスCOVID-19の影響で中止となりました名古屋・大阪・東京・神奈川の振替公演スケジュールが決定致しました！

     

    振替の日程は、下記になります。

     

    ＜愛知・名古屋ダイアモンドホール＞

    2月29日（土）　→　振替 7月30日（木）開場18:00／開演19:00

     

    ＜大阪・なんばHatch＞

    3月7日（土）　→　振替8月13日（木）開場18:00／開演19:00

     

    ＜東京・LINE CUBE SHIBUYA（渋谷公会堂）＞

    3月21日（土）　→　振替 7月11日（土）開場17:00／開演18:00

     

    ＜神奈川・神奈川県民ホール＞

    4月4日（土）　→　振替8月21日（金）開場18:00／開演19:00


     

    既にお手元に届いておりますチケットは、振替公演にて整理番号並びに座席はそのまま有効となりますので振替公演日まで大切に保管お願い致します。半券を切られてるチケットは無効となりますのでご注意ください。

     

    振替公演のスケジュールのご都合が合わないお客様に関しましては、払い戻しの対応をさせていただきますので下記の各公演の払戻し情報をご確認お願いします。

     

    【　振替公演に関しまして　】

    下記の日程よりファンクラブ先行を実施致します。

    ファンクラブ先行受付 : 4月1日 12:00　〜　4月7日 23:59

     

    ※LINE CUBE SHIBUYA公演は、売り切れ公演だったため払戻しされるお客様がいらっしゃらなければ先行受付対象外となりますので予めご了承ください。

    ※払い戻されたお座席の再販売となるため良席・良番をお約束するものではございません。
